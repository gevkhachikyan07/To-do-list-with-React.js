# To-do List (React)

This is a simple To-do List application built with React (Redux).
It works with localStorage and also has night/day mode.


---

##  Features

- Add tasks
- Delete tasks
- Edit tasks
- Delete all tasks with one but
- Mark tasks as done
- LocalStorage persistence (tasks saved after refresh)
- Simple UI

---

##  Tech Stack

- React (Redux)
- JavaScript
- HTML/CSS
